Introduction

Ce TP nous a permis de découvrir l'utilité du langage PHP qui permet de gérer la partie côté serveur. L'objectif de se TP était de pouvoir échanger avec la base de données pour aller récuperer le catalogue de prix ainsi que de gérer la connexion des clients avec en première partie l'enregistrement et l'autre la vérification si le client existe dans la base de données.
